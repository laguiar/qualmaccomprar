Qual Mac Comprar?

Comprar um novo computador não é uma tarefa muito fácil, mesmo pra quem usa os computadores da Apple, são muitas combinações possíveis.

Este guia tenta exatamente facilitar essa escolha, partindo da melhor relação Performance x Preço, em cada linha de Mac disponível.

Todos computadores tiveram a escolha dos itens (processador/mem/disco) visando aumentar a performance padrão.
